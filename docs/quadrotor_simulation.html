Quadrotor Simulation Pluto Notebook

This is a placeholder for the Pluto notebook exported to static HTML.

To view the interactive notebook, please open `quadrotor_simulation.jl` in a Pluto.jl environment.

If you wish to host the static HTML version, please manually export this notebook from Pluto.jl:

  1. Open `quadrotor_simulation.jl` in Pluto.
  2. From the Pluto UI, go to "Export" and select "Static HTML".
  3. Save the generated `quadrotor_simulation.html` file into the `docs/` directory of this repository.
